Negotiation Analysis: An Introduction

Wheeler's note is the standard analytical primer whose vocabulary the Basics slides compress. It builds the negotiator's core toolkit, starting with the BATNA (best alternative to a negotiated agreement) — the walk-away option that sets your real source of power and your reservation price, the worst deal you would still accept.

From the two sides' reservation prices it derives the ZOPA (zone of possible agreement): the range, if any, in which a deal beats both parties' alternatives. No ZOPA means no deal is rational; a wide ZOPA means the question is how the surplus is split.

It then separates distributive bargaining (claiming a fixed pie) from integrative bargaining (creating value by trading across issues that the parties value differently). The discipline is to analyse interests, alternatives and trade-offs before the conversation, so that tactics in the room serve a prepared strategy rather than improvisation.

Key takeaways

  • BATNA is your walk-away alternative and the true source of bargaining power
  • Reservation price = the worst deal you would still accept
  • ZOPA = the overlap of both sides' reservation prices; no overlap, no rational deal
  • Distributive bargaining claims a fixed pie; integrative bargaining creates value via trades
  • Analyse interests, alternatives and trade-offs before entering the room
This is a synthesis written for revision, not the original text.
Source: M. A. Wheeler, ‘Negotiation Analysis: An Introduction’, HBS Note 801-156
